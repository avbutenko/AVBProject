# Controls
## PC
- WASD - Movement
- E - Interact
- F - Use Candle/FlashLight
- R - Use currently active Perk 
- Q - switch items from QuickInvenory
- LeftShift - use item from QuickInvenory

## Android
Use on screen buttons
